Group Values
------------

Our collaboration is built on:

- **Openness**: Share ideas and results freely
- **Respect**: Value diverse perspectives
- **Rigor**: High standards for methods and validation
- **Support**: Help colleagues succeed
- **Integrity**: Honest communication and attribution


Collaborative Projects
----------------------

Working on a group project?

1. **Planning**: Discuss scope and goals
2. **Communication**: Regular updates
3. **Code**: Use shared repository
4. **Data**: Organize systematically
5. **Review**: Get feedback before publishing
6. **Attribution**: Credit all contributors


Mentoring
---------

**Finding a Mentor**

New group members are matched with mentors.

**Expectations**

- Regular one-on-one meetings
- Guidance on research topics
- Career development advice
- Introduction to lab community

**For Mentors**

- Support mentee's research
- Help with technical issues
- Discuss career goals
- Provide feedback and encouragement


Publication Process
-------------------

**Writing a Paper**

1. **Outline & Discussion**: Get input from co-authors
2. **First Draft**: Write initial manuscript
3. **Internal Review**: Group provides feedback
4. **Revisions**: Update based on feedback
5. **Final Review**: Have experts review
6. **Submission**: Submit to target journal

**Authorship**

- Discuss authorship order early
- Use CRediT for contribution clarity
- Ensure all contributors are listed
- Follow journal guidelines

Conflict Resolution
-------------------

If conflicts arise:

1. Try to resolve directly with parties involved
2. Bring to group lead if needed
3. Mediation available if requested
4. Commitment to fair and respectful resolution
