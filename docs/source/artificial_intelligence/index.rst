- Automating complex decision-making
- Learning from unstructured data
- Natural language understanding and generation
- Computer vision and image analysis
- Drug discovery and materials design
- Autonomous optimization and control

AI vs. Machine Learning
------------------------

- **Machine Learning**: Learning from data to make predictions
- **AI**: Broader field including ML, reasoning, planning, knowledge representation
- **Deep Learning**: Subset of ML using neural networks
- Often these terms overlap in practice
