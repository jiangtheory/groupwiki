AI Resources
============

Learning Materials
------------------

**Online Courses & MOOCs**

- **Coursera**: AI for Everyone, Deep Learning Specialization
- **edX**: AI Fundamentals, Machine Learning
- **Fast.ai**: Practical Deep Learning for Coders
- **Stanford CS221**: AI Principles and Techniques
- **Stanford CS224N**: NLP with Deep Learning
- **MIT OpenCourseWare**: AI and machine learning courses


**Books**

- "Artificial Intelligence: A Modern Approach" by Russell & Norvig (comprehensive textbook)
- "Deep Learning" by Goodfellow, Bengio, Courville
- "Reinforcement Learning: An Introduction" by Sutton & Barto
- "Attention Is All You Need" (transformer paper)
- "A Brief History of Artificial Intelligence" by Michael Wooldridge


**Key Papers**

- Transformer architecture (Vaswani et al., 2017)
- Attention mechanisms (Bahdanau et al., 2014)
- Deep neural networks (LeCun et al., 1998)
- AlphaGo (Silver et al., 2016)
- AlphaFold (Jumper et al., 2020)
- BERT (Devlin et al., 2018)


Datasets for AI Research
------------------------

**Text Datasets**

- **Common Crawl**: Massive web crawl
- **Wikipedia Dumps**: Encyclopedia data
- **arXiv**: Scientific papers
- **PubMed**: Biomedical literature

**Image Datasets**

- **ImageNet**: 14M labeled images
- **COCO**: Object detection dataset
- **STL-10**: Image recognition

**Domain-Specific**

- **Kaggle Datasets**: Competition datasets
- **Hugging Face Datasets**: Ready-to-use datasets
- **Scientific Data**: Domain repositories


Pre-trained Models
------------------

**Language Models**

- **GPT-4**: Advanced language model
- **Claude**: Anthropic's language model
- **Llama 2**: Meta's open-source model
- **BERT**: Bidirectional encoder
- **T5**: Text-to-Text Transfer Transformer

Available on Hugging Face model hub.

**Vision Models**

- **ResNet**: Image classification
- **ViT**: Vision Transformer
- **CLIP**: Vision-language models
- **DALL-E**: Image generation

**Specialized Models**

- **AlphaFold**: Protein structure
- **ESMFold**: Fast protein folding
- **Stable Diffusion**: Image generation
