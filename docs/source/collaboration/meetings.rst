Presentation Guidelines
-----------------------

**Preparing Your Talk**

- Outline your main message first
- Include background context
- Show preliminary/final results
- Anticipate questions
- Practice beforehand


**Slide Tips**

- Title slide: Title, name, date
- Motivation: Why does this matter?
- Background: Context and related work
- Methods: What did you do?
- Results: What did you find?
- Discussion: What does it mean?
- Questions/Next steps


**During Presentation**

- Speak clearly and confidently
- Make eye contact with audience
- Allow time for questions
- Be open to feedback
- Thank the audience


**Feedback Culture**

- Ask clarifying questions
- Give constructive feedback
- Focus on the science
- Be respectful and supportive
- Follow up privately if needed

Meeting Etiquette
-----------------

**Do's**

- ✓ Arrive on time
- ✓ Come prepared
- ✓ Participate actively
- ✓ Ask questions
- ✓ Support colleagues


**Don'ts**

- ✗ Multitask on computer
- ✗ Phone usage
- ✗ Interrupt others
- ✗ Dominate discussion
- ✗ Dismiss others' ideas

Virtual Meeting Tips
-------------------

**Technology**

- Test setup 5 min before
- Good lighting and background
- Clear audio (mute when not speaking)
- Use video when possible
- Have backup plan for technical issues


**Engagement**

- Use features (raise hand, reactions)
- Share screen when helpful
- Keep chat organized
- Minimize distractions


Cancellations & Changes
-----------------------

**If You Need to Cancel**

- Notify early (24 hours if possible)
