Getting Started
===============

Welcome to the research group! This guide will help you get up to speed.


First Week Checklist
--------------------

- [ ] Meet your mentor/supervisor
- [ ] Set up your computing environment
- [ ] Get access to lab resources and servers
- [ ] Review lab guidelines and safety procedures
- [ ] Join communication channels (Slack, email lists, etc.)
- [ ] Attend group meetings and seminars
- [ ] Read group overview materials


Computing Environment Setup
---------------------------

**System Requirements**

- Compute node access (contact IT)
- Python 3.8+ installation
- Git for version control
- IDE/Editor (VS Code, PyCharm, etc.)

**Essential Tools**

- See :doc:`resources/software` for detailed software setup
- MD Simulation tools: GROMACS, LAMMPS, AMBER, etc.
- ML frameworks: PyTorch, TensorFlow, scikit-learn
- Visualization: VMD, Pymol, Matplotlib, etc.

**Getting Started with Group Code**

1. Clone the group repositories
2. Install dependencies: ``pip install -r requirements.txt``
3. Run tests to verify setup: ``pytest``
4. Check the :doc:`collaboration/guidelines` for code standards
