Getting Involved
----------------

Interested in a project?

1. Read the project description
2. Contact the project lead
3. Discuss how you can contribute
4. Get started!

See :doc:`../getting_started` for onboarding.


Project Opportunities
---------------------

Looking for a project to join?

- Check status of projects above
- Talk to group leads
- Consider your interests and skills
- See :doc:`../team` for mentors


Proposing New Projects
----------------------

Have a research idea? Here's how:

1. **Initial Idea**: Discuss with mentor
2. **Literature Review**: What's existing work?
3. **Proposal**: Write brief proposal
4. **Group Discussion**: Present to group
5. **Feedback**: Refine based on feedback
6. **Execution**: Get started!
