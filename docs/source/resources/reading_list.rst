Topic-Specific Reading Paths
----------------------------

**Just Starting in ML?**

1. Watch: 3Blue1Brown linear algebra (YouTube)
2. Read: The Hundred-Page ML Book
3. Course: Fast.ai or Google ML Crash Course
4. Try: Kaggle competitions

**Want to Do Deep Learning?**

1. Course: Fast.ai or Coursera DL Specialization
2. Book: Deep Learning by Goodfellow et al.
3. Implement: Simple networks in PyTorch
4. Read: Key papers from :doc:`../research/papers`

**Interested in Scientific ML?**

1. Background: Classic MD books, physics
2. Papers: Key papers on ML potentials
3. Tools: LAMMPS, OpenMM, DeepChem
4. Project: Build your own ML potential

**Researching Protein Structure?**

1. Review: AlphaFold paper
2. Try: ESMFold or ColabFold
3. Learn: Structure biology fundamentals
4. Research: Read related papers
